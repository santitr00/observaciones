# Libro de Actas Digital 📜🚀

> Una plataforma moderna, intuitiva y segura diseñada para optimizar la gestión, el registro y el resguardo de actas institucionales, consorcios y organizaciones.

## 💡 Sobre el Proyecto

El **Libro de Actas Digital** transforma el tedioso proceso manual de gestión de actas en un flujo de trabajo 100% digital, centralizado y eficiente. Adiós a las hojas impresas perdidas y a las revisiones manuales interminables.

### Características Principales

*   **Gestión Centralizada:** Creá, registrás y adjuntá documentación de respaldo a cada acta en un entorno seguro.
*   **Buscador Inteligente:** Encontrá cualquier dato al instante filtrando por palabras clave o rangos de fecha.
*   **Exportación Universal:** Descargá actas individuales o múltiples en formato PDF con un solo clic, ideal para auditorías o asesores legales/contables.
*   **Autonomía de Usuarios:** Sistema granular de roles y permisos (**Administrador, Consejo, Auditor**) para gestionar altas y bajas sin depender de soporte.

## 🛠️ Stack Tecnológico

*   **Backend:** [Flask](https://flask.palletsprojects.com/) (Python)
*   **ORM:** [SQLAlchemy](https://www.sqlalchemy.org/)
*   **Base de Datos:** [MySQL](https://www.mysql.com/) / [PostgreSQL](https://www.postgresql.org/)
*   **Frontend:** HTML5, CSS3, JavaScript (Bootstrap / Tailwind)
